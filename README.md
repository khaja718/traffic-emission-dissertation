# Traffic Emission Prediction Using Simulation and Machine Learning

This repository contains the code, experiments, and outputs for an MSc dissertation on traffic emission prediction using simulation, spatio-temporal graph neural networks, and knowledge distillation.

## Structure
- `notebooks/` experimental notebooks
- `src/` reusable Python modules
- `data/` raw and processed datasets
- `results/` figures, models, and plots

## Workflow
1. Network setup
2. Simulation and dataset generation
3. Vehicle density experiments
4. ST-GNN teacher training
5. Knowledge distillation to student model
